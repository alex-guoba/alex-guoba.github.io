golang pprof

以常见的one-gouroutine-per-connection模式来介绍pprof的用法

测试数据

CPU Profiling

  • flat：给定函数上运行耗时
  • flat%：同上的 CPU 运行耗时总比例
  • sum%：给定函数累积使用 CPU 总比例
  • cum：当前函数加上它之上的调用运行总耗时
  • cum%：同上的 CPU 运行耗时总比例

10000个connection时测试数据

Heap profiling

  • -inuse_space：分析应用程序的常驻内存占用情况(default)
  • -alloc_objects：分析应用程序的内存临时分配情况
